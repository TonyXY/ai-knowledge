# Few-Shot / Zero-Shot <DifficultyBadge level="beginner" />

**Zero-Shot** 和 **Few-Shot** 是提示词工程中最基础的两个概念，它们描述的是你在 Prompt 中**是否提供示例**。

---

## Zero-Shot：不给示例，直接提问

Zero-Shot 意为"零示例"。你直接问问题，不给任何示范，AI 凭自身知识回答。

```text
把以下句子翻译成英文：今天天气真好。
```

```text
请判断这句话的情感倾向是正面还是负面："这个产品太难用了，我要退款！"
```

**适用场景**：任务简单明确、AI 有充足训练知识、你不需要特定风格或格式。

**局限性**：对于格式要求高或风格特殊的任务，Zero-Shot 的效果往往不够好。

---

## Few-Shot：先给示例，再提问题

Few-Shot 意为"少量示例"。你在 Prompt 中先展示 2-3 个**输入→输出**示例，再让 AI 处理你的实际输入。

### 示例：文本分类

```text
请判断以下句子的情感倾向（正面/负面/中性）。

示例 1：
输入：物流太慢了，等了一周才到！
输出：负面

示例 2：
输入：质量不错，性价比很高，会回购。
输出：正面

示例 3：
输入：收到了，还没用，暂时不做评价。
输出：中性

现在请判断：
输入：包装完好，但颜色和图片差太多了。
输出：
```

通过 3 个示例，AI 不仅学会了分类任务，还学会了你的输出格式（只写"正面/负面/中性"）。

### 示例：风格模仿

```text
请用以下风格回复客户投诉：

示例 1：
客户：你们发货太慢了！
回复：非常抱歉给您带来不便！我已查询您的订单 #12345，目前正在配送中，预计明天送达。为表歉意，已为您申请一张 20 元优惠券，请注意查收。

示例 2：
客户：收到的商品有划痕！
回复：非常抱歉！请您拍照发送给我们，我立刻为您安排换货或退款，您选择哪种方式都可以。您的满意对我们很重要。

现在请回复：
客户：你们的 App 老是闪退，根本用不了！
回复：
```

## 对比效果表

| 维度 | Zero-Shot | Few-Shot |
|------|-----------|----------|
| Prompt 长度 | 短 | 较长 |
| 输出格式控制 | 弱 | 强 |
| 风格一致性 | 不稳定 | 稳定 |
| 适用任务 | 简单任务 | 复杂/格式要求高的任务 |
| 示例数量 | 0 | 通常 2-5 个 |

---

## Few-Shot 最佳实践

1. **示例要典型** — 选择有代表性的案例，覆盖不同情况
2. **格式要一致** — 所有示例使用相同的输入→输出结构
3. **数量要适中** — 2-3 个通常足够，超过 5 个边际收益递减
4. **先易后难** — 示例从简单到复杂排列
5. **最后放实际输入** — 把你要处理的内容放在所有示例之后
