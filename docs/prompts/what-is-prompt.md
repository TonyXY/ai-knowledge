# 什么是 Prompt <DifficultyBadge level="beginner" />

**Prompt（提示词）** 就是你给 AI 的输入指令。无论是一句话、一段文字，还是一个完整的结构化文档，都是 Prompt。

> 核心理念：**输入的质量，决定输出的质量。**

---

## Prompt 的三种角色

在一次完整的对话中，Prompt 通常包含三种角色：

| 角色 | 说明 | 谁写的 |
|------|------|--------|
| **System Prompt** | 系统级指令，设定 AI 的行为基调 | 开发者 / 你（API 调用时） |
| **User Prompt** | 你直接发给 AI 的问题或指令 | 你 |
| **Assistant** | AI 之前的回复（多轮对话时传递） | AI |

这三种角色共同构成了 [LLM](/terms/llm) 的上下文。

---

## 好 Prompt vs 坏 Prompt

同样是让 AI 写一份产品介绍，Prompt 的质量差异巨大：

### ❌ 坏 Prompt

```text
写个产品介绍。
```

AI 不知道：什么产品？给谁看？多长？什么风格？结果大概率是泛泛而谈。

### ✅ 好 Prompt

```text
你是一位产品营销专家。我们需要为一款名为"智记"的 AI 笔记应用写一段产品介绍。
目标用户是大学生和白领，文案风格轻松活泼，字数 150 字左右。
重点突出三个功能：语音转文字、AI 自动摘要、跨设备同步。
```

提供的信息越多，AI 的输出越精准。

---

## 好 Prompt 的三个要素

1. **清晰** — 指令不模糊，AI 不需要猜测你的意图
2. **具体** — 有具体要求和约束条件，而不是开放式的"随便写"
3. **有上下文** — 告诉 AI 背景信息、目标用户、使用场景

---

## 对比示例：翻译任务

### ❌ 模糊指令

```text
翻译成英文。
```

### ✅ 清晰指令

```text
请将以下中文翻译成英文，要求：
- 保持原文语气（正式商务风格）
- 使用英式拼写
- 翻译后附上 1-2 句翻译说明

原文：我们致力于为客户提供一站式数字化解决方案。
```
