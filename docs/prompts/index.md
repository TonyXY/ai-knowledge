# 💬 提示词工程

提示词（Prompt）是你与 AI 沟通的桥梁。一个好的提示词，能让 AI 从"答非所问"变成"精准输出"。

本模块按三个层次组织，建议从基础篇开始，再到进阶技巧，最后用场景模板实战练习。

<DifficultyBadge level="beginner" /> 无需前置知识 · <DifficultyBadge level="intermediate" /> 建议先掌握基础篇

---

## 🟢 基础篇

从这里入门，建立对提示词的基本认知。

<div class="card-grid">
  <a href="./what-is-prompt" class="card">
    <h3>什么是 Prompt <DifficultyBadge level="beginner" /></h3>
    <p>理解 Prompt 的本质——你给 AI 的指令。介绍 System / User / Assistant 三部分结构，以及好 Prompt 的标准。</p>
  </a>
  <a href="./principles" class="card">
    <h3>清晰指令五原则 <DifficultyBadge level="beginner" /></h3>
    <p>写出好 Prompt 的五条核心原则：明确任务、提供上下文、指定格式、给出示例、约束边界。每条原则配对比示例。</p>
  </a>
  <a href="./role-prompting" class="card">
    <h3>角色设定 <DifficultyBadge level="beginner" /></h3>
    <p>让模型扮演特定角色（专家、教师、编辑等），显著提升特定领域的输出质量。</p>
  </a>
  <a href="./few-shot" class="card">
    <h3>Few-Shot / Zero-Shot <DifficultyBadge level="beginner" /></h3>
    <p>Zero-Shot 不给示例直接问，Few-Shot 先给 2-3 个示例再提问。理解两者的区别和适用场景。</p>
  </a>
</div>

## 🟡 进阶技巧

掌握这些技巧，你的提示词水平将超越大多数使用者。

<div class="card-grid">
  <a href="./system-prompt" class="card">
    <h3>System Prompt 设计 <DifficultyBadge level="intermediate" /></h3>
    <p>System Prompt 是预设的系统级指令，影响模型全程行为。好的设计 = 角色 + 规则 + 输出约束 + 边界条件。</p>
