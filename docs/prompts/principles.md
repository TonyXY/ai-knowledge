# 清晰指令五原则 <DifficultyBadge level="beginner" />

写好 Prompt 并不需要什么神秘技巧。遵循以下五条原则，你的提示词质量就能大幅提升。

---

## 原则一：明确任务目标

说清楚你要 AI 做什么，而不是让它猜。

### ❌ 不好

```
帮我写点东西。
```

### ✅ 好

```
帮我写一封离职邮件，语气礼貌得体，感谢公司的培养，不涉及任何负面内容。
```

**技巧**：用动词开头——"写一封"、"总结以下"、"将以下翻译成"、"分析这段数据"。

---

## 原则二：提供足够上下文

告诉 AI 背景信息：谁在用、为什么用、什么场景下用。

### ❌ 不好

```
解释一下 Vue 的响应式原理。
```

### ✅ 好

```
我的读者是刚学前端 3 个月的新手，请用通俗易懂的方式解释 Vue 的响应式原理，
用生活中的类比来帮助理解，避免使用专业术语。
```

**技巧**：至少回答三个问题——谁在看？需要什么程度的答案？什么场景下使用？

---

## 原则三：指定输出格式

告诉 AI 你想要的输出结构和样式。

### ❌ 不好

```
给我讲讲 Python 和 JavaScript 的区别。
```

### ✅ 好

```
请用表格对比 Python 和 JavaScript 的主要区别，包含以下维度：
类型系统、运行环境、主流用途、语法特点、社区生态。
表格后附一段 50 字以内的总结。
```

**常用格式**：表格、列表、JSON、Markdown、分步骤说明、摘要+详情。

---

## 原则四：给出示例（Few-Shot）

先给 AI 看 1-3 个你想得到的输出样例，它就能模仿你的风格。

### ❌ 不好

```
把这段文字改写成简洁版本。
```

### ✅ 好

```
请把以下句子改写成简洁版本，参考示例格式：

示例：
原文：由于天气原因，原定于明天举行的户外活动将会被推迟到下周进行。
改写：明天活动因天气推迟至下周。

现在请改写：由于系统正在进行维护升级，在此期间用户可能无法正常登录账户。
```

更多示例技巧见 [Few-Shot / Zero-Shot](./few-shot)。

---

## 原则五：约束边界

明确告诉 AI **不要做什么**，和告诉它做什么同样重要。

### ❌ 不好

```
推荐一些 Python 学习资源。
```

### ✅ 好

```
推荐 5 个适合初学者的免费 Python 学习资源。
要求：
- 不包括视频教程（只要文章/文档/交互式平台）
- 每个资源附 1 句话推荐理由
- 不要推荐付费课程
- 优先中文资源
```

---

## 五原则速查卡

| 原则 | 一句话 | 关键词 |
|------|--------|--------|
| 明确任务 | 说清楚做什么 | 动词开头 |
| 提供上下文 | 说清楚为谁做 | 读者、场景、深度 |
| 指定格式 | 说清楚怎么呈现 | 表格、列表、Markdown |
| 给出示例 | 让 AI 看到样例 | 输入→输出示例 |
| 约束边界 | 说清楚不要什么 | 限制条件、排除项 |
