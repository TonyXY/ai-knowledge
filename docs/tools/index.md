## 🤔 怎么选？

| 场景 | 推荐工具 |
|------|----------|
| 日常对话、写作 | ChatGPT / DeepSeek |
| 长文档分析 | Claude / Kimi / Gemini |
| 代码开发 | Cursor / Copilot |
| 预算有限 | DeepSeek（API）/ Kimi（免费） |
| 画图 | Midjourney（艺术）/ DALL·E（易用）/ SD（免费） |

每个工具页都标注了难度等级 —— <DifficultyBadge level="beginner" /> 表示无门槛上手，<DifficultyBadge level="intermediate" /> 需要一定基础。
