# Kimi

## 一句话定位

**长上下文国产助手**——200 万字上下文窗口，一键总结整本书不是梦，还支持强大的联网搜索。

## 基本信息

| 项目 | 详情 |
|------|------|
| 公司 | 月之暗面 (Moonshot AI) |
| 价格 | 免费 |
| 主要模型 | Kimi 最新模型 |
| 上下文窗口 | 200 万字（约 400K tokens） |
| 首发时间 | 2023 年 10 月 |

## 核心功能

### 📚 超长上下文（200 万字）
这是 Kimi 最突出的特点。你可以：
- 上传一整本书，让它总结核心内容
- 上传几百页的 PDF 论文，逐章分析
- 上传法律合同，逐条审查风险条款
- 上传项目文档，对比多个版本差异

### 🌐 联网搜索
Kimi 的联网搜索能力非常强，可以搜索网页、公众号文章等中文互联网内容。搜索结果质量高，引用清晰。

### 📄 文件处理
支持上传 TXT、PDF、Word、Excel、PPT、图片等多种格式。上传后可以直接针对文件内容提问。

### 🔧 浏览器插件
提供 Chrome/Edge 浏览器插件，可以在浏览网页时随时调用 Kimi 进行总结、翻译等。

## 优势

- **超长上下文**：200 万字的处理能力在同类产品中名列前茅
- **完全免费**：目前没有任何付费计划，所有功能免费使用
- **联网搜索**：中文互联网搜索质量高，尤其适合需要查找国内信息
- **中文友好**：国产模型，中文理解和生成自然流畅
- **无使用限制**：免费版没有明显的次数限制

## 不足

- 不支持图像生成
- API 生态不如 OpenAI 丰富
- 代码能力弱于 Claude 和 DeepSeek
- 复杂推理深度不如 Claude 3.5 Sonnet
- 不开源，无法私有化部署

## 适合谁？

- ✅ 需要处理超长文档的用户
- ✅ 中文场景为主的用户
- ✅ 学生和研究人员（论文阅读、文献整理）
- ✅ 需要联网搜索国内信息的用户
- ✅ 预算为零的个人用户
- ⚠️ 需要高强度代码开发的用户建议搭配 Cursor 或 Claude
