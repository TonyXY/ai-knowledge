# Windsurf

## 一句话定位

**Codeium 出品的 AI IDE**——Cascade 模式可以自主完成多步骤编程任务，与 Cursor 各有所长。

## 基本信息

| 项目 | 详情 |
|------|------|
| 公司 | Codeium (Windsurf) |
| 价格 | 免费版 / Pro $15/月 / Teams $35/人/月 |
| 支持模型 | GPT-5 / Claude 4 / Claude 3.5 Sonnet / 自有模型 |
| 平台 | macOS / Windows / Linux |
| 基于 | VS Code |

## 核心功能

### 🌊 Cascade 模式
Windsurf 的特色功能。与传统对话式 AI 不同，Cascade 会：
- 分析你的代码库
- 理解你的意图
- 自主执行多步操作（创建文件、修改代码、运行命令）
- 在过程中主动反馈进度

简单说：你可以告诉它"帮我创建一个 React 登录页面"，它会自己创建组件文件、写代码、装依赖、甚至启动项目。

### 🔄 AI Flow
Cascade 在执行任务时会保持"AI Flow"状态——它记得之前做了什么，能连贯地完成一系列操作，而不是每次对话都从零开始。

### 📝 多文件编辑
支持同时修改多个文件，能理解跨文件的引用关系，避免"改了 A 忘记改 B"的问题。

### 🏎️ Supercomplete
代码补全功能，支持整行、整函数补全，质量接近 Cursor。

## 优势

- **Cascade 自主性**：能自动执行多步骤任务，减少手动操作
- **价格更低**：Pro 版 $15/月，比 Cursor 便宜
- **免费版功能强**：免费版包含 Cascade 基础功能和代码补全
- **自有模型**：Codeium 有自研模型，不完全依赖第三方

## 不足

- 生态和社区不如 Cursor 成熟
- 部分高级功能（如长时间 Cascade 会话）有限额
- 中文场景的表现偶尔不稳定
- 自主执行时可能出现意料之外的操作，需要监督

## 适合谁？

- ✅ 喜欢"下指令后让 AI 自己干"风格的开发者
- ✅ 希望降低 AI 工具成本的用户
- ✅ 想尝试 Cursor 之外的替代方案
- ⚠️ 对自主执行不放心的用户建议多用 Cursor 的手动确认模式
