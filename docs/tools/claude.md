# Claude

## 一句话定位

**长文本最强的 AI 助手**——200K 上下文窗口 + 深度推理能力，特别适合需要处理大量信息的复杂任务。

## 基本信息

| 项目 | 详情 |
|------|------|
| 公司 | Anthropic |
| 价格 | 免费版 / Pro $20/月 / Team $25/人/月 |
| 主要模型 | Claude 3.5 Sonnet / Claude 3 Opus |
| 上下文窗口 | 200K tokens |
| 首发时间 | 2023 年 3 月 |

## 核心功能

### 📄 超长上下文（200K tokens）
一次可以处理约 15 万英文单词或一整本书的内容。你可以上传完整的技术文档、学术论文、甚至整本小说，Claude 都能完整"阅读"并回答相关问题。

### 💻 代码能力
Claude 3.5 Sonnet 在代码生成和理解方面表现顶尖，尤其擅长：
- 分析大型代码库
- 重构复杂逻辑
- 生成完整项目代码
- 解释技术概念

### 📊 文档分析
上传 PDF、Word、Excel、PPT 等文件，Claude 可以提取信息、总结要点、对比分析、生成报告。

### 🎨 Artifacts 功能
对话中可以直接生成并预览交互式网页、SVG 图表、React 组件等，所见即所得。

## 优势

- **推理深度**：在复杂逻辑推理、数学证明、代码分析方面非常出色
- **安全性**：Anthropic 以"宪政 AI"理念著称，输出更安全、更负责任
- **长文本**：200K 上下文在同类产品中名列前茅
- **写作质量**：文字产出自然流畅，中文表达能力不错

## 不足

- 不支持图像生成
- 联网搜索需手动开启，不如 Kimi/DeepSeek 方便
- 免费版使用限额较严格
- 多模态能力不如 ChatGPT（图像理解有，但不支持生成）

## 适合谁？

- ✅ 长文档分析（论文、合同、技术规范）
- ✅ 深度推理场景（数学、逻辑、代码分析）
- ✅ 编程开发（代码生成、重构、Review）
- ✅ 对 AI 安全性有要求的场景
- ⚠️ 需要图像生成或频繁联网搜索的用户建议搭配其他工具
