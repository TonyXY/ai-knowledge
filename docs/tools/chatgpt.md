# ChatGPT

## 一句话定位

**最通用的 AI 对话助手**——无论你想聊天、写代码、翻译、分析文档还是画图，ChatGPT 都是最先值得一试的选择。

## 基本信息

| 项目 | 详情 |
|------|------|
| 公司 | OpenAI |
| 价格 | 免费版 / Plus $20/月 / Pro $200/月 |
| 主要模型 | GPT-5 / GPT-4o / GPT-4o-mini / o1 / o3 |
| 上下文窗口 | 256K tokens (GPT-5) / 128K tokens (GPT-4o) |
| 首发时间 | 2022 年 11 月 |

## 核心功能

### 💬 多模态对话
支持文字、图片、文件（PDF、Word、Excel 等）输入。你可以上传一张截图让它分析、上传一份 PDF 让它总结、上传 Excel 让它生成图表。

### 💻 代码能力
GPT-4o 在代码生成、Debug、代码解释方面表现优异，是许多开发者的首选助手。支持几乎所有主流编程语言。

### 🔌 GPTs 与插件
可以创建自定义 GPTs（特定领域的专用助手），也可以调用联网搜索、DALL·E 画图、代码解释器等内置工具。

### 🎙️ 语音对话
手机 App 支持语音输入和实时语音对话，体验接近真人交流。

## 优势

- **生态最大**：用户量最多，教程最丰富，第三方集成最广泛
- **插件丰富**：GPTs 商店里有大量专用助手，覆盖写作、编程、教育等场景
- **多模态强**：文字 + 图像 + 语音 + 文件，一个平台全搞定
- **持续迭代**：OpenAI 更新频率快，模型能力不断提升

## 不足

- 免费版有使用额度限制，高峰期可能降级到低配模型
- Plus 用户也有一定的 GPT-4o 调用频率限制
- 部分高级功能（如 o1-pro 深度推理）仅限 Pro 版
- 数据隐私需注意：默认情况下对话可能被用于模型训练

## 适合谁？

- ✅ 日常使用、通用场景的首选
- ✅ 写作、翻译、学习辅导
- ✅ 需要多模态能力的用户（图文混合场景）
- ✅ 想要丰富插件生态的用户
- ⚠️ 对数据隐私有极高要求的用户建议使用企业版或 API
