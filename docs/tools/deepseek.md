# DeepSeek

## 一句话定位

**国产最强开源模型**——性能媲美 GPT-4，价格却只有几十分之一，是预算敏感场景的最佳选择。

## 基本信息

| 项目 | 详情 |
|------|------|
| 公司 | 深度求索 (DeepSeek) |
| 价格 | 免费 / API $0.27/M tokens（输入） |
| 主要模型 | DeepSeek-V3 / DeepSeek-R1 |
| 上下文窗口 | 128K tokens |
| 首发时间 | 2023 年 |

## 核心功能

### 💬 对话
DeepSeek 网页版和 App 完全免费，没有使用次数限制。支持联网搜索，可以获取最新信息。

### 💻 代码
代码能力是 DeepSeek 的强项之一。无论是生成代码、Debug、还是解释复杂算法，效果都很出色。

### 🧠 推理
DeepSeek-R1 是专门的推理模型，在数学、编程、逻辑推理等任务上表现惊人。它会在回答前展示完整的思考过程。

### 🔌 API 接入
API 价格极低：输入 $0.27/M tokens，输出 $1.10/M tokens。对比 GPT-4o（输入 $2.50/M tokens），成本优势非常明显。

## 优势

- **性价比极高**：API 成本是 GPT-4o 的 1/10 左右，免费版无限使用
- **开源可私有化**：模型权重完全开源，可以在自己服务器上部署
- **推理能力强**：R1 模型在数学和编程领域达到顶尖水平
- **中文友好**：国产模型，中文理解和生成都很自然

## 不足

- 生态不如 OpenAI 丰富（没有 GPTs 商店那样的生态）
- 不支持图像生成
- 多模态能力较弱（不支持原生图像理解）
- 高峰期响应速度可能变慢

## 适合谁？

- ✅ 预算有限的个人开发者
- ✅ 需要 API 批量调用的场景
- ✅ 需要私有化部署的企业
- ✅ 中文对话为主的用户
- ✅ 数学推理、代码分析等专业场景
- ⚠️ 需要图像生成或多模态能力的用户建议搭配 DALL·E 或 Midjourney
