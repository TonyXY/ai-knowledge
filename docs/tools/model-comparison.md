## 各模型详解

### OpenAI 系列

#### GPT-5

| 项目 | 说明 |
|------|------|
| **定位** | OpenAI 新一代旗舰多模态模型，全面超越 GPT-4o |
| **核心能力** | 文本、图像、音频、视频理解，Function Calling、代码，Agent 原生支持 |
| **突出优势** | 256K 上下文、多模态能力全面提升、Agent 工作流最成熟、推理能力大幅增强 |
| **主要不足** | 价格比 GPT-4o 更贵（$4/$16），Pro 版深度模式仅限高端订阅 |
| **适合场景** | 复杂多模态任务、Agent 自动化、对质量要求最高的生产场景 |

#### GPT-4o

| 项目 | 说明 |
|------|------|
| **定位** | OpenAI 旗舰多模态模型（上一代），通用场景的全能选手，**仍广泛使用** |
| **核心能力** | 文本、图像理解、音频(实时语音)、Function Calling、代码 |
| **突出优势** | 多模态能力成熟、生态最完善(GPTs/Plugin)、工具调用稳定、性价比高 |
| **主要不足** | 推理深度不如 o1/o3 和 Claude 系列，已被 GPT-5 全面超越 |
| **适合场景** | 通用对话、多模态应用、Function Calling 工作流、预算敏感的生产环境 |

#### GPT-4o-mini

GPT-4o 的轻量版。**速度快、价格低**，适合高频调用和简单任务。能力在大多数日常场景中足够用。

#### o1

OpenAI 的**推理模型**，内部会进行"思考"再回答。在数学、编程竞赛、复杂逻辑推理上显著优于 GPT-4o，但速度慢、价格贵。适合需要深度推理的场景。**o3** 是最新一代推理模型，推理深度进一步提升，在竞赛级数学和编程中接近人类专家水平。

### Anthropic 系列

#### Claude 4

| 项目 | 说明 |
|------|------|
| **定位** | Anthropic 新一代旗舰模型，代码能力全面领先 |

#### Claude 3.5 Sonnet

| 项目 | 说明 |
|------|------|
| **定位** | 编程+长文本的最强选择之一，**仍广泛使用** |

Anthropic 的轻量级模型——速度快、价格低（$0.80/$4），适合高频调用和简单任务。在代码补全、快速问答等场景中表现优秀，性价比远超 Claude 3 Opus。

## 如何选择

### 按预算

```
预算极低 ───────────────────────────────────── 不设上限
 Gemini 2.5 Flash  DeepSeek-V4  GPT-4o-mini  Claude 4  o3
  $0.10/M          $0.27/M     $0.15/M      $4.00/M  $20.00/M
```

### 按任务

```
代码开发    → Claude 4 ≈ GPT-5 > Claude 3.5 Sonnet > DeepSeek-V4
深度推理    → o3 > o1 ≈ DeepSeek-R1 > Claude 4
长文档处理  → Gemini 2.5 Pro (2M) > Claude 4 > GPT-5
中文场景    → DeepSeek-V4 ≈ Qwen3 > GPT-5
视觉理解    → GPT-5 ≈ Gemini 2.5 Pro > Claude 4
性价比      → DeepSeek-V4 ≈ Gemini 2.5 Flash > GPT-4o-mini
自部署      → DeepSeek-V4 > Qwen3 > Llama 4
```

### 最佳组合推荐

**个人开发者**：DeepSeek-V4（日常）+ Claude 3.5 Sonnet（代码）
**企业用户**：GPT-5（通用/Agent）+ DeepSeek-V4（高并发）
**预算敏感**：Gemini 2.5 Flash（超长文本）+ DeepSeek-V4（通用）
**私有化部署**：DeepSeek-V4 + Qwen3
