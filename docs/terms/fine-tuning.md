# 微调 (Fine-tuning)

你有一个通晓万物的通用模型，但你需要它擅长某个特定领域——微调就是为此而生的技术。

## 一句话解释

微调（Fine-tuning）是在已预训练好的通用模型基础上，用少量特定领域的数据继续训练，让模型在特定任务上表现更好，同时保留原有的通用能力。LoRA 是目前最流行的高效微调方法。

## 通俗类比

**通用模型** = 一个医学院刚毕业的全科医生，什么病都能看一点，但都不精。

**微调** = 派他去心血管科轮训 3 个月——他保留原有的医学基础（不会变成只会看心脏），但对心血管疾病特别擅长。

- 不需要从头读 8 年医学（重新训练大模型）
- 只需要 3 个月专科培训（少量领域数据）
- 成本低、见效快

## 技术定义

微调的核心是**迁移学习（Transfer Learning）**。数学上，就是在预训练权重的基础上，在特定数据集上继续执行梯度下降。

**全量微调（Full Fine-tuning）** 更新模型的所有参数，效果好但成本高（需要与训练相当的硬件）。

**参数高效微调（PEFT）** 是更实用的方案：

| 方法 | 原理 | 特点 |
|------|------|------|
| **LoRA** | 在原始权重旁插入可训练的低秩矩阵 | 只训练极少参数（<1%），效果接近全量微调 |
| **Adapter** | 在 Transformer 层间插入小型网络 | 插件式，可堆叠多个适配器 |
| **Prompt Tuning** | 只训练一段可学习的 "软提示" | 参数量最小，适合简单任务 |

微调的典型应用场景：
- **领域定制**：让通用模型精通医疗、法律、金融等专业领域
- **风格对齐**：让模型输出符合特定语气、格式或品牌调性
- **指令跟随**：提升模型遵循特定类型指令的能力
- **知识注入**：补充训练数据截止日期之后的新知识
