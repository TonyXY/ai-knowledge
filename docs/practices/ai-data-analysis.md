# 用 AI 做数据整理

面对一张上千行的 Excel 表格不知道该从哪下手？让 AI 来帮你。从理解数据结构到生成分析报告，不用写一行代码，也能做专业级的数据分析。

## 准备工作

- 一份 CSV 或 Excel 数据文件
- 一个能上传文件的 AI 工具（ChatGPT Plus、Claude Pro、DeepSeek 等）
- 明确你分析的目标（比如"找出销售趋势""分析用户行为"）

---

## 第 1 步：上传数据，让 AI 理解结构

先把数据喂给 AI，让它理解每一列是什么意思。这一步很关键，AI 理解了数据含义，后续分析才会准确。

### 示例 Prompt

```text
我上传了一份公司 2025 年 Q1-Q4 的销售数据 CSV 文件。
请帮我理解这份数据的结构：

1. 一共有多少行、多少列？
2. 每一列的名称和数据类型是什么？
3. 有哪些列可能存在空值？
4. 根据列名，你判断这份数据是用来做什么的？

请用中文回答，列出分析结果。
```

AI 会输出数据的概览。如果你对它理解的列含义不满意，及时纠正：

```text
"amount" 这一列不是金额，是订单数量。请更正你的理解。
```

---

## 第 2 步：让 AI 做数据清洗

原始数据通常很"脏"——有重复行、空值、格式不一致。让 AI 帮你洗数据。

### 示例 Prompt

```text
请用 Python 对这份数据做清洗，具体任务：

1. 删除完全重复的行
2. 检查 "price" 列是否都是数字格式，如果不是请修正
3. "date" 列统一成 YYYY-MM-DD 格式
4. 如果某行的 "customer_name" 为空，填充为 "未知客户"
5. 删除 "price" 为 0 或负数的异常行

请输出清洗后的数据摘要（行数变化、修改了哪些值），以及清洗用的 Python 代码。
```

AI 会输出清洗代码和统计结果：

```python
import pandas as pd

df = pd.read_csv('sales_data.csv')

# 删除重复行
before = len(df)
df = df.drop_duplicates()
print(f"删除了 {before - len(df)} 行重复数据")

# 统一日期格式
df['date'] = pd.to_datetime(df['date']).dt.strftime('%Y-%m-%d')

# 填充空值
df['customer_name'] = df['customer_name'].fillna('未知客户')

# 删除异常行
df = df[df['price'] > 0]

print(f"清洗后剩余 {len(df)} 行数据")
```

把代码复制到本地 Python 环境运行，或者直接在 ChatGPT/Claude 的代码执行环境中跑。

---

## 第 3 步：让 AI 做分析

数据干净了，开始真正的分析。带着问题去问 AI，比让它"随便分析一下"效果好得多。

### 示例 Prompt

```text
基于刚才清洗后的销售数据，请帮我做以下分析：

1. 按季度统计总销售额，观察趋势（上升还是下降？）
2. 找出销售额前 5 的产品
3. 分析客户复购率（同一客户购买多次的比例）
4. 检查有没有异常大单（单笔金额超过平均值 3 倍标准差的记录）

每个分析结果请给出你的解读，不只是数据，还要说明"这意味着什么"。
```

AI 会逐一计算并给出结论。对于不确定的判断，你可以追问：

```text
你提到 Q3 销售额下降了 12%，这算是正常波动还是异常信号？
结合完整年份的数据来判断一下。
```

---

## 第 4 步：让 AI 生成可视化方案和报告

分析做完了，要把结果呈现出来。告诉 AI 你的受众是谁，它会推荐合适的图表类型。

### 可视化建议 Prompt

```text
我需要把这些分析结果做成一份报告，面向公司管理层（非技术人员）。

请为下面的每个分析结果推荐一种图表类型：
1. 季度销售额趋势 → ___
2. 前 5 产品对比 → ___
3. 客户复购率 → ___
4. 异常订单分析 → ___

对每个推荐，说明为什么选这种图（比如"趋势用折线图，因为强调时间变化"）。
```

### 生成报告 Prompt

```text
基于以上全部分析结果，请帮我写一份数据分析报告。
要求：
- 标题：2025 年度销售数据分析报告
- 结构：摘要 → 整体趋势 → 产品表现 → 客户分析 → 问题发现 → 建议
- 语言：简洁专业，每个观点配数据支撑
- 篇幅：800 字左右
```

---

## 关键技巧总结

1. **先让 AI 理解数据**：告诉它每列的含义和数据类型，后续分析才准确
2. **清洗后再分析**：脏数据直接分析，结论不可靠
3. **带具体问题去问**："帮我分析"不如"帮我看销售额趋势、找前 5 产品、检查异常值"
4. **让 AI 解读，不只是计算**：要求它对每个结果给出"这意味着什么"
5. **敏感数据脱敏后再上传**：客户姓名、手机号等隐私信息先处理掉
