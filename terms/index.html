📖 AI 术语 ​

按难度分为三级，建议按顺序学习，也可以直接跳到感兴趣的词条。

🟢 入门 无需前置知识 · 🟡 进阶 需了解基础概念 · 🔴 高级 需一定实践经验

📜 建议先了解背景

在学习具体术语之前，推荐先阅读 AI 发展历程 ——了解每个概念是在什么背景下诞生的，会让学习更有方向感。

🟢 入门术语（10 个） ​

这些是 AI 领域最基础的概念，零基础从这里开始。

AI / ML / DL

人工智能、机器学习、深度学习的关系与区别

大语言模型 (LLM)

GPT、Claude 等模型背后的核心技术

Token

AI 处理文本的最小单位，也是计费单位

上下文窗口

模型一次能"记住"的最大信息量

Prompt

你给 AI 的指令——输入的质量决定输出的质量

多模态

同时处理文本、图像、音频等多种数据类型

参数 (Parameters)

模型的"神经元"数量，决定模型能力上限

训练 / 推理

模型的学习阶段和使用阶段的区别

微调 (Fine-tuning)

让通用模型适应特定任务的技术

AI 伦理与法规

偏见、隐私、版权、全球法规——负责任的 AI

🟡 进阶术语（10 个） ​

理解这些，你就超越了大多数 AI 使用者。

Transformer

现代大模型的基石架构

自注意力机制

Transformer 的核心创新

RAG

检索增强生成——让 AI 连接外部知识

Embedding

将文本转化为向量，让机器理解语义

向量数据库

存储和检索向量的专用数据库

Function Calling

让 AI 调用外部工具和 API

Agent

能自主规划、执行任务的 AI 智能体

AI Skills

Agent 的模块化可复用专业技能包

MCP

AI 与外部工具/数据源交互的开放协议

Prompt Caching

缓存重复 Prompt 片段，降低成本和延迟

🔴 高级术语（5 个） ​

深入模型底层的进阶概念。

MoE

混合专家模型——高效扩展模型规模

思维链 (CoT)

引导模型逐步推理的技术

RLHF

基于人类反馈的强化学习

KV Cache

加速推理的关键优化技术

DPO

无需奖励模型的偏好对齐方案，RLHF 的简化替代
